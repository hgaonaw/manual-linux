Head y Tail

Descripción

Los comandos head y tail se utilizan para mostrar las primeras o últimas líneas de un archivo, respectivamente. Son herramientas muy útiles para examinar grandes archivos de texto, permitiendo inspeccionar rápidamente sus primeras o últimas entradas sin tener que abrir el archivo completo.

Opciones

  • -n número → Especifica el número de líneas que se mostrarán, ya sea al principio con head o al final con tail.
  • -f → En tail, permite seguir mostrando las nuevas líneas que se van agregando a un archivo en tiempo real (útil para logs).

Ejemplos

# Ejemplo 1: Mostrar las primeras 10 líneas de un archivo
head archivo.txt

# Ejemplo 2: Mostrar las primeras 5 líneas
head -n 5 archivo.txt

# Ejemplo 3: Mostrar las últimas 10 líneas
tail archivo.txt

# Ejemplo 4: Mostrar las últimas 3 líneas
tail -n 3 archivo.txt

# Ejemplo 5: Seguir un archivo en tiempo real (logs)
tail -f archivo.log
