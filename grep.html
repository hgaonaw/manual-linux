Comando GREP

Comando

grep

Descripción

El comando grep se utiliza en sistemas Linux y Unix para buscar texto dentro de archivos. Permite encontrar líneas que coincidan con un patrón específico, ya sea una palabra, frase o expresión regular.

Sintaxis

grep [opciones] patrón [archivo]

Opciones

  • -i: Ignora mayúsculas y minúsculas en la búsqueda.
  • -r: Busca de forma recursiva dentro de directorios.
  • -n: Muestra el número de línea donde se encuentra la coincidencia.

Ejemplos

# Buscar la palabra "error" en un archivo
grep error archivo.txt

# Buscar sin distinguir mayúsculas/minúsculas
grep -i error archivo.txt
