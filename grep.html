GREP

Descripción

El comando grep se utiliza para buscar texto dentro de archivos. Permite encontrar palabras o patrones específicos y mostrar las líneas donde aparecen.

Opciones

  • -i → Ignora mayúsculas y minúsculas.
  • -n → Muestra el número de línea.
  • -r → Busca de forma recursiva en directorios.

Ejemplos

# Buscar una palabra en un archivo
grep "texto" archivo.txt

# Ignorar mayúsculas/minúsculas
grep -i "texto" archivo.txt

# Mostrar número de línea
grep -n "texto" archivo.txt

Errores comunes

# Error: no usar comillas
grep texto largo archivo.txt

# Correcto
grep "texto largo" archivo.txt

# Error: olvidar -i
grep "hola" archivo.txt

# Correcto
grep -i "hola" archivo.txt
