CHMOD

Descripcion

El comando chmod en Linux se usa para cambiar los permisos de acceso (lectura, escritura y ejecución) de archivos y directorios. Permite decidir quién puede hacer qué con un archivo: el propietario, el grupo o otros usuarios. Por ejemplo, puedes dar permiso de ejecución a un script o restringir el acceso a un archivo privado.

Opciones

  • -R: aplica los cambios de forma recursiva a todos los archivos y carpetas dentro de un directorio.

  • -v: muestra un mensaje detallado de cada cambio realizado.

  • -c: muestra solo los cambios cuando realmente se modifica algún permiso.

  • -f: modo silencioso, no muestra errores si ocurren.

    • Ejemplo De uso:

    chmod u+x script.sh

    Esto añade permiso de ejecución solo al usuario propietario.
