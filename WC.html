WC

Descripción

El comando wc en Linux se utiliza para contar líneas, palabras y caracteres de un archivo o entrada de texto, permitiendo obtener rápidamente información básica sobre su contenido y tamaño.

Opciones

  • -l: muestra solo el número de líneas.

  • -w: muestra solo el número de palabras.

  • -m: muestra el número de caracteres.

    • Ejemplo De uso:

    wc archivo.txt

    Este comando muestra el número de líneas, palabras y bytes que contiene el archivo archivo.txt.
