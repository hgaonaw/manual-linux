Cut

Descripción

El comando cut se utiliza para cortar secciones específicas de cada línea de un archivo de texto o de la entrada estándar. Este comando permite seleccionar columnas delimitadas por caracteres o posiciones, extrayendo únicamente la parte que interese al usuario. Es útil cuando se necesita procesar grandes volúmenes de datos en formato tabular.

Opciones

  • -d delimitador → Especifica el delimitador que se utiliza para separar las columnas en cada línea. Por defecto, es un tabulador.
  • -f campos → Selecciona los campos (columnas) que se desean mostrar, donde los campos se numeran a partir de 1.
  • -c rangos → Selecciona caracteres en posiciones específicas de cada línea.

Ejemplos

# Ejemplo 1: Mostrar la primera columna de un archivo separado por comas
cut -d "," -f 1 archivo.csv

# Ejemplo 2: Mostrar la segunda y tercera columna
cut -d "," -f 2,3 archivo.csv

# Ejemplo 3: Mostrar un rango de columnas (de la 1 a la 3)
cut -d "," -f 1-3 archivo.csv

# Ejemplo 4: Extraer caracteres del 1 al 5 de cada línea
cut -c 1-5 archivo.txt

# Ejemplo 5: Extraer desde el carácter 3 hasta el final
cut -c 3- archivo.txt
