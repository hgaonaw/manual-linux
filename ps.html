PS

Descripción

El comando ps se utiliza en sistemas Linux y Unix para mostrar información sobre los procesos que se están ejecutando en el sistema. Permite ver el identificador del proceso (PID), el usuario, el uso de CPU y memoria, entre otros datos.

Opciones

  • -e → Muestra todos los procesos del sistema.
  • -f → Muestra información detallada (formato completo).
  • -u → Muestra los procesos de un usuario específico.

Ejemplos

# Mostrar todos los procesos
ps -e

# Mostrar procesos con formato completo
ps -ef
