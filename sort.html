Sort

Descripción

El comando sort ordena las líneas de un archivo o de la entrada estándar en orden ascendente o descendente. Este comando es útil para organizar información y buscar patrones o datos ordenados de forma alfabética o numérica. El orden por defecto es ascendente, pero se pueden especificar otras opciones como el orden descendente o personalizar la forma en que se realiza la ordenación.

Opciones

  • -r → Ordena las líneas en orden descendente (de Z a A, en lugar de A a Z).

  • -n → Realiza una ordenación numérica, en lugar de alfabética.

  • -k clave → Especifica la columna (o clave) por la cual se va a ordenar el texto, útil cuando se trabaja con datos tabulares.

Ejemplos

# Ejemplo 1: Ordenar un archivo alfabéticamente
sort archivo.txt

# Ejemplo 2: Ordenar en orden descendente
sort -r archivo.txt

# Ejemplo 3: Ordenar números correctamente
sort -n numeros.txt

# Ejemplo 4: Ordenar por la segunda columna (separada por espacios)
sort -k 2 archivo.txt

# Ejemplo 5: Ordenar por columna con delimitador específico (por ejemplo, coma)
sort -t "," -k 2 archivo.csv
