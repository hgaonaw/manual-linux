FIND

Descripción

El comando find se utiliza para buscar archivos y directorios dentro del sistema. Permite filtrar por nombre, tamaño, fecha de modificación, permisos y ejecutar acciones sobre los resultados.

Opciones

  • -name → Busca por nombre de archivo.
  • -size → Filtra por tamaño del archivo.
  • -mtime → Filtra por fecha de modificación.
  • -exec → Ejecuta un comando sobre los resultados.

Ejemplos

# Buscar archivos .txt modificados en el último día
find /ruta -name "*.txt" -mtime -1

# Buscar archivos con permisos 644
find /ruta -perm 644

# Buscar archivos mayores a 10MB
find /ruta -size +10M

# Borrar archivos .log
find /ruta -name "*.log" -exec rm {} \;

Errores comunes

# Error: olvidar comillas
find /ruta -name *.txt

# Correcto
find /ruta -name "*.txt"

# Error: usar mal el tamaño
find /ruta -size 10M

# Correcto
find /ruta -size +10M

# Error: olvidar escapar ;
find /ruta -name "*.log" -exec rm {} ;

# Correcto
find /ruta -name "*.log" -exec rm {} \;
